# 真人口型示范拍摄指南

v0.5 的目标不是创建泛视频库，而是为《枪火》01:05-01:17 的关键粤语口型准备真人示范素材。

每条 demo 都必须回答：

> 这个口型示范帮助用户唱准《枪火》的哪一句、哪个音？

## 1. Shooting Setup

- Phone camera is enough.
- Use front camera or rear camera with tripod.
- Shoot in vertical 9:16 first for short-video reuse.
- Also allow 16:9 later for desktop product.
- Use soft front light.
- Keep background clean.
- Record in quiet room.
- Mouth and lower face must be clearly visible.

## 2. Required Shots Per Sound

For each key sound, record:

1. 正面慢速读
2. 正面节奏读
3. 侧面慢速读（可选）
4. 放回歌词句子
5. 放回《枪火》节奏

## 3. Clip Length

每条 4-8 秒。
不要讲长课。
只示范一个动作。

## 4. Naming Convention

Use:

```text
/public/demos/gunfire/
  ngo5-front-slow.mp4
  ngo5-front-rhythm.mp4
  gwong-front-slow.mp4
  gwong-front-rhythm.mp4
```

Posters can mirror the same name:

```text
/public/posters/gunfire/
  ngo5-front-slow.jpg
  ngo5-front-rhythm.jpg
```

## 6. Performance Notes

- 示范要“清楚”，不是“夸张”。
- 先慢速，再原节奏。
- 不要用力喊。
- Rap 的港味来自咬字、节奏和气场，不是大声。

## 7. 英文近似入口原则

有些粤语音普通话里没有，可以用英文近似音帮助学员找到口腔动作。

但必须明确：

- 英文不是标准粤语发音。
- 英文只是入口。
- 每条 demo 都要说明“英文和粤语的细微区别”。
- 最后必须回到粤语目标音和《枪火》句子。

推荐慢速示范顺序：

```text
普通话习惯/错误口型
→ 英文近似入口
→ 粤语目标音
→ 放回《枪火》句子
```

## 8. Product Mapping

Every clip should map to:

- one MouthCoach card,
- one or more provided lyric lines,
- one target sound,
- one application point inside the Boss Segment.

Example:

```text
ngo5 正面慢速
解锁：第 1 句 / 第 3 句的「我」
重点：鼻音起音，不要读成 wo5
```
